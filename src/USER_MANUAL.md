# คู่มือการใช้งานระบบคลังและจัดการคำสั่งองค์กร (PEA Prompt Library User Manual)

ยินดีต้อนรับสู่ **PEA Prompt Library** เครื่องมือศูนย์กลางระดับองค์กรสำหรับการสืบค้น จัดเก็บ ปรับปรุง และแบ่งปันวิศวกรรมคำสั่ง (Prompt Engineering) อย่างเป็นระบบ ออกแบบมาสำหรับวิศวกร บุคลากร และแผนกไอทีภายในของการไฟฟ้าส่วนภูมิภาค (PEA) เพื่อเพิ่มเสถียรภาพ ประสิทธิภาพ และความปลอดภัยในการใช้งานปัญญาประดิษฐ์ (AI) 

---

## 🎨 การออกแบบและสอดคล้องตามมาตรฐาน (PEA Design System Compliance)
ระบบแอปพลิเคชันได้รับการออกแบบตาม **Visual Identity ของ PEA Design System (`@pea-ds/*`)** อย่างสมบูรณ์:
- **Brand Identity colors**: ใช้โทนสีประจำองค์กรเฉดสีม่วงครามโตรมาติก (Fuchsia/Magenta Brand colors) รหัสสีประจำสถาปัตยกรรม (เช่น `#cf07aa` และ `#a80689`) สอดคล้องในทุกหัวข้อ ปุ่ม และสถานะต่าง ๆ
- **Default Grounding Spine**: ข้อมูลตัวหนังสือและสีกรอบอิงเฉดสีเทาสุขุม (Grounding Neutral Slate Ramp) 
- **Typography**: จับคู่ตระกูลภาษาไทยเพื่อแสดงผลอย่างเป็นทางการผ่านฟอนต์ **IBM Plex Sans Thai** ร่วมกับฟอนต์สากลระดับพรีเมียม **Inter** และสไตล์โค้ดด้วยฟอนต์ **JetBrains Mono**
- **Sizing Density**: ปรับสมดุลเว้นระยะขอบซ้าย-ขวาอย่างสมมาตร งดงาม สบายตากับหน้าจอเดสก์ท็อปและสมาร์ทโฟน

---

## 🚀 ฟีเจอร์หลักและการใช้งาน (Core Features & Workflows)

### 1. แผงควบคุมและคลังแสดงผล Prompt (Prompt Directory Dashboard)
หน้าแรกรวบรวมการจัดการ Prompt ทั้งหมดไว้เพื่อความสะดวกสูงสุด โดยสามารถเลือกดูข้อมูลได้ตามสองสไตล์:
*   **Grid View (มุมมองการ์ด)**: แสดงการ์ด Prompt พร้อมคำอธิบายย่อย แท็กหมวดหมู่ และเครื่องมือ AI ที่แนะนำแบบมีมิติวิชวลสวยงาม
*   **Table View (มุมมองตาราง)**: จัดเรียงแบบกะทัดรัด (Compact) เพื่อดูรหัส เจ้าของ วันที่อัปเดต และสถิติการดึงใช้งานในรูปแบบตารางที่เหมาะกับการวิเคราะห์ข้อมูลเร็ว
*   **สถิติภาพรวม (Telemetry Counter)**: ด้านข้างแสดงยอดจัดเก็บคำสั่งทั้งหมด ยอดสะสมการสืบค้น และสถิติการใช้งาน Favorite ของตัวเอง

---

### 2. ระบบสืบค้นและตัวกรองอัจฉริยะ (Real-Time Search & Multiple Filters)
สืบค้น Prompt ที่ตรงกับงานวิศวกรรมและงานธุรการของคุณได้อย่างทันทีบนแถบค้นหาแถวบนสุด:
*   **แบบกรองพิมพ์ด่วน (Real-time Search)**: ค้นหาคำสำคัญจากชื่อหัวข้อ รายละเอียดเนื้อหา แท็ก หรือแม้แต่ชื่อผู้สร้างสรรค์ (Creator)
*   **ตัวกรอง AI Tools ที่แนะนำ (Recommended Models)**: เลือกสืบค้น Prompt ที่ออกแบบมาเฉพาะสำหรับโมเดล เช่น GPT-4, Claude-3.5, Gemini-1.5, Google Maps API หรือโมเดลอื่น ๆ แบบเจาะจง
*   **ตัวกรองบทบาทหน้าที่ (Prompt Types)**:
    *   *User Prompt*: คำสั่งป้อนข้อมูลทั่วไปของผู้ใช้งาน
    *   *System Prompt*: คำสั่งกำหนดวิสัยทัศน์หรือบทบาทจำลองของโมเดล
    *   *Instruction*: คำแนะนำหรือกฎเฉพาะทางแบบเป็นขั้นตอน
    *   *Skill*: ชุดทักษะที่มีบล็อกคำสั่งสลับเปลี่ยนซับซ้อน
*   **การเรียงลำดับ (Sorting Options)**: เรียงข้อมูลตามความสดใหม่ของการอัปเดต (Recently Updated), ความนิยมการใช้งานสูงสุด (Most Popular) หรือเรียงตามลำดับอักษรหัวข้อ (A-Z)

---

### 3. บานหน้าต่างตรวจสอบคำสั่งและแบบฟอร์มกรอกตัวแปร (Interactive Prompt Inspector)
เมื่อคลิกที่ Prompt Card ใด ๆ ด้านขวามือจะปรากฏบานหน้าต่าง **Prompt Inspector** ทันที เพื่อให้คุณควบคุมตรวจสอบโครงสร้างคำสั่งได้โดยละเอียดโดยไม่ต้องเปลี่ยนหน้า:
*   **ตรวจสอบ Metadata**: แสดงสิทธิ์การเปิดเผย (Public Organization / Private Draft), ข้อมูลผู้ออกแบบ (By PEA Staff) และโมเดลที่แนะนำให้ใช้ร่วมกัน
*   **ตัวแปรทดแทนแบบกรอกสด (Live Variable Substitution Form)**:
    *   บานหน้าต่างจะวิเคราะห์หาตัวแปรที่อยู่ในรูปแบบซิงแทกซ์ `{ชื่อตัวแปร}` ภายใน Prompt ทั้งหมดโดยอัตโนมัติ 
    *   ระบบจะแปลงตัวแปรเหล่านั้นออกมาเป็นช่องป้อนข้อมูล (Input Fields) ให้คุณกรอกข้อความตามตัวแปรของคุณเอง
    *   **Live Preview**: ในส่วนแสดงข้อความคำสั่ง บล็อกตัวหนังสือจะได้รับการแทนค่า (Substitution) แบบสด ๆ ทันควันตามที่คุณกดป้อนผ่านคีย์บอร์ดโดยอัตโนมัติ!
*   **ตัวเลือกการคัดลอก**:
    1.  **คัดลอกพร้อมใช้ (Copy Substituted)**: ดึงเนื้อหาข้อความ Prompt ที่นำสิ่งที่คุณกรอกแทนที่ตัวแปรเสร็จสิ้นแล้วลงสู่คลิปบอร์ด เพื่อนำไปวางและส่งงานต่อบนซอฟต์แวร์อื่นได้ทันที
    2.  **นำเข้าเทมเพลต (Use Template)**: ถ่ายโอนโครงร่าง Prompt ไปยังส่วนจำลองแผงทดสอบคำสั่ง

---

### 4. ระบบเปิดใช้งานด่วนและการบันทึกแบบเร่งด่วน (Quick Save Drawer)
กรณีที่คุณค้นพบหรือสร้างคำสั่งเด่นในระหว่างการทำงาน และต้องการจัดเก็บบันทึกเข้าระบบ PEA Prompt Library ไว้อย่างรวดเร็ว:
*   คลิกที่ปุ่มสีม่วง **Quick Save** ที่อยู่บริเวณส่วนเมนูด้านบนขวา
*   บานลิ้นชัก (Drawer) จะเลื่อนพับออกมาจากทางขวาสุด คลาวด์ซิงโครไนซ์พร้อมกรอกข้อมูลพื้นฐานอย่างรวดเร็ว (ชื่อหัวข้อ, รายละเอียดโดยย่อ, เนื้อหา Prompt)
*   เมื่อบันทึกข้อมูลจะถูกนำไปติดตั้งลงบนฐานข้อมูลเซิร์ฟเวอร์ทันควัน ให้คุณและเพื่อนร่วมงานเข้าถึงต่อได้โดยไม่ต้องเสียเวลาสร้างหลายขั้นตอน

---

### 5. แผงสร้างสรรค์คำสั่งและบริหารตัวแปรเวอร์ชันเต็ม (Advanced Prompt Creator & Editor)
สำหรับนักวิจัยหรือผู้ดูแลวิศวกรรมการป้อนคำสั่ง (Prompt Engineers) ของ PEA คุณสามารถสร้าง Prompt ที่มีมาตรฐานสูงได้ผ่านแผงหน้าต่างใหญ่:
*   **Multi-Model Recommended Selection**: กำหนดเครื่องมือเป้าหมาย AI ได้มากกว่าหนึ่งเครื่องมือ (เช่น แนะนำใช้ได้ทั้ง Gemini Pro และ GPT-4) ผ่านกล่องค้นหารองรับการลบแท็กแบบ Badge บาจสีชมพูองค์กรใช้งานง่าย
*   **Modular Blocks Setup**: สามารถเพิ่มและแยกกลุ่มบล็อกคำสั่ง เช่น บล็อกสถานะของระบบ (System Instruction), ทิศทางคำแนะนำ (Context Block), และตัวแปรคำถามหลัก (User Query Block) เพื่อปรับโครงร่างคำสั่งให้เป็นมาตรฐานสากล
*   **แผงจัดการตัวแปร (Variables Generator)**: แฟรกเมนต์ตัวช่วยสร้างฟิลด์ตัวแปรที่จำเป็น และเพิ่มคำแนะนำในการตอบคำถาม เพื่อชี้แจงให้แก่ผู้ใช้งานท่านอื่นที่เข้ามานำคำสั่งส่วนนี้ไปใช้ต่อ

---

## 🛡️ สิทธิ์การเข้าใช้งานและความมั่นคงปลอดภัย (Security & Access Control Rules)
เพื่อป้องกันเหตุความเสียหาย และรักษามาตรฐานข้อมูลทางการค้าภายในของการไฟฟ้าส่วนภูมิภาค:
- **ล็อกอินรับสิทธิ์ใช้งาน (Internal Authentication)**: บุคลากรสามารถล็อกอินเข้าระบบด้วยระบบความปลอดภัยองค์กรเพื่อรับสถานะในการสร้างคำสั่งของตนเอง
- **สิทธิ์ความเป็นเจ้าของ (Defensive Ownership Enforcement)**: ปุ่มสำหรับแก้ไขลบและแก้ไขข้อมูล Prompt จะจำกัดและแสดงผลเฉพาะผู้เขียนตัวจริง (Owner) และทีมผู้ดูแลระบบอาวุโส (Admin) เท่านั้น โดยระบบเซิร์ฟเวอร์หลังบ้านมีการตรวจสอบ CORS, ป้องกัน SQL Injection ด้วย Type-safe ชั้นหิน เพื่อปกป้องความลับความมั่นคงปลอดภัยขั้นสูงสุด
